A python RAG chatbot created using streamlit and langchain(openAI models).

Step 1:
We can create a new chat by giving the chat name.
Existing chats appear below the new chat input box.

Step 2:
Upload the document we want to use as a source.
We can also give a web page link to be used as a source.

Step 3:
After successful upload of the document or link, ask any question within the scope of the source.
The bot will successfully answer the question.
